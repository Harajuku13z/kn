Générateur de Favicons

Instructions

Pour générer les favicons :

  1. Copiez le contenu du fichier SVG ci-dessous
  2. Allez sur RealFaviconGenerator
  3. Collez le SVG dans l'outil
  4. Téléchargez le package complet
  5. Placez les fichiers dans le dossier public/favicon/
